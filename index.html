Frequently asked questions

Is my PDF uploaded to a server?

No. Your file and its password are processed entirely on your device using WebAssembly. Nothing is uploaded — there is no server to receive it. You can confirm this yourself by opening your browser’s network tab: no file ever leaves your device.

Can this crack or recover a password I don’t know?

No. This tool only removes a password you already know. You must enter the correct password; it does not guess, brute-force, or recover unknown passwords. If you’ve lost the password, this tool cannot open the file.

Does it work offline?

Yes. Once the page has loaded, all processing happens locally in your browser, so it keeps working without an internet connection.

Which devices and browsers are supported?

It runs in any modern browser — Chrome, Firefox, Safari, and Edge — on Windows, macOS, Linux, Android, and iPhone or iPad. No app or installation is needed.

What kind of PDF password does it remove?

It removes the open password (the one your PDF reader prompts you for). Once removed, the new PDF opens freely with no password prompt and no usage restrictions.

Is there a file size limit?

There is no fixed limit. Because everything runs in your browser, very large files are constrained only by your device’s available memory rather than any upload cap.

Is it free and open source?

Yes. The tool is free to use and the source code is open, so anyone can inspect exactly what it does. It is powered by the MuPDF library compiled to WebAssembly.
